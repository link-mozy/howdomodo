# 1. 프로젝트 개요

- 보고싶은 영화를 CGV, 메가박스, 롯데시네마 중에서 가장 밀집도가 낮은 영화관으로 영화를 편하게 볼 수 있도록 제공하는 서비스.
- 영화 전후로 시간이 남거나 식사를 해야하는 경우 나이와 성별에 따른 활동을 추천해준다.
- 기본 기능으로 현재 상영중인 영화의 후기를 분석하여 추천해준다.
